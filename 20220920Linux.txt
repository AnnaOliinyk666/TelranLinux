В нескольких словах дайте, пож., ответ на след. вопросы:



1.Зачем нужен GIT и какие задачи решают с его помощью?

ответ: для управления версиями

Что такое репозиторий GIT и где он находится?

ответ: это все файлы версии которых находятся под контролем. 
Можно создать из любой папки на компьютере (локальный) 
и привязать и выгружать на гит (удаленный)

Для чего используется файл .gitignore?

ответ: для указания в нем файлов которые должны быть проигнорированны

Что такое "коммит"?

ответ: контрольная точка состояния проекта на текущий момент времени


Укажите базовые команды в рамках работы с GIT:

git init
git add
git status
git commit
git rm
git branch
git push
git remote add



Настройка аккаунта локального пользователя. 
git config
git config --global user.name
git config --global user.email

Создание репозитория.
git init

Добавление файлов в индекс и выполнение коммита.
git add .
git commit -m '___'

Просмотр журнала коммитов.
git log


Ответы выгрузите текстовым файлом на GitHub.
